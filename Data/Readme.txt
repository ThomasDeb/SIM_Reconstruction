This folder contains the following data:
    - Acqdata.tif : 9-images raw SIM data from the OpenSIM Package (https://github.com/LanMai/OpenSIM)
    - patterns.tif : corresponding illumination patterns generated with the function PatternsFromFairSimXml.m (see folder PatternsFromFairSIM/ for an example of use) given the parameters estimated with FairSIM (https://www.fairsim.org/)
    - PSF.tif : associated PSF 
    - FairSimParameters.xml : xml file (containing patterns parameters) obtained with the FairSIM ImageJ plugin on the Acqdata.tif images.
    - object.tif : sample from "K. Wicker, O. Mandula, G. Best, R. Fiolka, R. Heintzmann, Phase optimisation for structured illumination microscopy, Opt. Expr. 21, 2032-2049, 2013."
